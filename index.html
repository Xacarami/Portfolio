Carrefour e-commerce com APIs

Feito em React, consome a API disponibilizada pelo Carrefour para criar o e-commerce, e uma API para encontrar os mercados mais próximos e seus produtos.
Foi desenvolvido durante um bootcamp, e então foi um dos 10 projetos escolhidos para serem apresentados aos programadores e ao RH do Carrefour
Foi usado axios, react-router-dom, e também hooks como useParams, useEffect, useState e useNavigate

SnapClean: Gerencie Arquivos

Feito em Java, esse app para Android acelera a limpeza de arquivos indesejados e liberação de espaço rápido. Basta selecionar uma pasta com imagens, vídeos, áudios ou outros tipos, e arrastar para um lado para excluir, e para o outro para manter. Seu código está fechado até o momento.
Meu primeiro app mobile. Foi utilizado a IDE Android Studio, e sua estrutura é feita de Java puro. A ajuda do ChatGPT foi útil para ajudar à usar funções já existentes do Android, como por exemplo o dispatchTouchEvent e onBackPressed.

Pesquisa no GitHub por API

Feito em React, ao pesquisar por um nickname, o perfil é procurado na API do GitHub. Ao encontrá-lo, mostrará informações como foto, empresa, e principalmente seus repositórios. No campo de repositórios, conforme você escreve, irá filtrando todos os repositórios de acordo com o título, descrição e linguagem principal.
